◆ Run()

void Brigerad::Application::Run ( )

Where all of the magic happens. During normal execution, this function never returns, for as long as the process window is open.

It iterates through all of the layers and ImGui layers contained in the application, handles the refreshing of window events (key press, mouse events, window closed/minimized, etc.), as well as handling the frame creation/rendering for ImGui.
